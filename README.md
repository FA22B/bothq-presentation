# bothq-presentation

Using Marp to create nice Markdown-generated slides.

# Installation

## CLI program

```bash
npm install --save-dev @marp-team/marp-cli
```

## VS Code Extension

[Marp for VS Code](https://marp.app/)

# Compile Command

```bash
npx @marp-team/marp-cli@latest --pdf presentation.md --allow-local-files
```
